Oracle Technologies:


ADF
In computing, Oracle Application Development Framework, usually called Oracle ADF , provides a commercial Java framework for building enterprise applications. It provides visual and declarative approaches to Java EE development.
SOA
Oracle SOA Suite provides a comprehensive suite of components for developing, securing, and monitoring service-oriented architecture ( SOA ).
Webcenter
Oracle WebCenter Portal is a web platform for creating intranets, extranets, composite applications, and self-service portals .
Weblogic
Oracle WebLogic Server is a Java EE application server currently developed by Oracle Corporation.
OSB
Oracle Service Bus transforms architectures by connecting, mediating, and managing interactions between services and applications.
